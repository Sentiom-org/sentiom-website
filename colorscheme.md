# Sentiom Color Scheme

## Primary Colors

### Primary Blue
- **Hex:** `#2E5EAA`
- **RGB:** `rgb(46, 94, 170)`
- **Usage:** Primary actions, CTAs, accents, links, banner backgrounds

### Dark Blue/Navy
- **Hex:** `#2E3478`
- **RGB:** `rgb(46, 52, 120)`
- **Usage:** Headings, primary text, hover states

## Backgrounds

### Off-White
- **Hex:** `#FAFAFA`
- **RGB:** `rgb(250, 250, 250)`
- **Usage:** Main background, cards

### Light Gray
- **Hex:** `#F6F7FB`
- **RGB:** `rgb(246, 247, 251)`
- **Usage:** Alternate section backgrounds, subtle backgrounds

### Light Accent
- **Hex:** `#E4E7F1`
- **RGB:** `rgb(228, 231, 241)`
- **Usage:** Borders, dividers, subtle accents

## Text Colors

### Primary Text
- **Hex:** `#374151`
- **RGB:** `rgb(55, 65, 81)`
- **Usage:** Body text, paragraphs

### Secondary Text
- **Hex:** `#6B7280`
- **RGB:** `rgb(107, 114, 128)`
- **Usage:** Subtitles, secondary information, labels

## Feature Cards (with images)

### Text on Dark Overlay
- **Color:** `rgba(246, 247, 251, 0.95-0.98)`
- **Shadow:** `0 1px 4px rgba(0, 0, 0, 0.3)`
- **Background overlay:** Black gradient from 20% to 60% opacity
